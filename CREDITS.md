# Credits

## Pixel-art assets

Pet sprites adapted from:
- **Kenney** ([kenney.nl](https://kenney.nl)) — *Tiny Dungeon* / *Tiny Creatures* packs (CC0, public domain).
  Conversion: `tools/png_to_grid.py` (PNG → 16×16 palette-mapped ASCII grid).

Custom-drawn (in-house): humanoid (16×20), Codex pets (`codex-tui` / `codex-exec` / `codex-vscode`),
furniture/decoration sprites, prop icons.

## Inspiration

- **Pixel Agents** (Pablo Delucca, VSCode extension) — pet-keeper metaphor for AI coding tools.
- **Pokémon Gen-2** silhouette economy at small scale.
- **Stardew Valley** NPC sprite proportions.

## Open-source dependencies

See `pyproject.toml` for Python dependencies. Frontend is dependency-free (single static HTML/CSS/JS).
